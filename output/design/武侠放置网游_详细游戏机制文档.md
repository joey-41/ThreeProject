# 武侠放置网游详细游戏机制文档

## 0. 文档定位

- 文档目标：将 `Document/【审阅稿】武侠放置网游_完整设计方案.docx` 的方向性方案，结合 `output/story/武侠放置网游_世界观_节点地图_背景故事线.md` 的世界观与地图叙事，整理为可供项目经理、系统策划、数值策划、客户端、服务端直接拆分研发任务的主机制文档。
- 版本范围：首发 MVP 的“中间扩展版”。高安与低安全开，危险区首发开放 `朔砂外原` 与 `伏龙渊域外围/前哨`。主线覆盖前三幕，第十章后的深层内容、`沉灯海隙` 与完整赛季战争保留接口。
- 设计原则：
  - 上班挂机，下班主动。
  - 市场由玩家驱动，系统只负责注入底层资源与银两回收。
  - PvP 必须存在，但整体安全感向生产玩家略微倾斜。
  - 休闲玩家不掉队，深度玩家有政治目标和高端收益。
  - 世界观不是包装层，而是玩法规则的解释器。

## 1. 产品北极星与首发目标

### 1.1 一句话定位

一款以“九河十三路”节点江湖为舞台的武侠放置网游：玩家白天经营生产、挂机清剿、调度交易，晚上参与副本、护镖、劫镖、帮战与区域政治，在可持续的玩家经济里创造自己的江湖传闻。

### 1.2 北极星体验

- 休闲玩家：每天 15-30 分钟也能持续成长，有稳定收菜感、剧情推进感、市场参与感。
- 深度玩家：每天 1-4 小时可以优化构筑、控制路线、参与帮战、垄断与反垄断，获得高风险高回报体验。
- 全体玩家：能感觉到“我在一个已经活着的江湖里做生意、冒险、结仇、结盟”。

### 1.3 首发成功标准

- D1 留存目标不低于 40%。
- D7 留存目标不低于 20%。
- D30 留存目标不低于 10%。
- 首月银两周净注入控制在 +1.5% 至 +3.5%。
- 高价值装备的日损毁率与日新增量比值稳定在 0.28 至 0.35。
- 任一联盟对危险区核心资源的持续控制占比不超过 45% 且不连续超过 7 天。

### 1.4 设计红线

- 不卖直接数值。
- 不卖产出翻倍、经验翻倍、战斗力保送。
- 不卖免损毁、免死亡、免风险特权。
- 免费玩家可完整体验核心成长、剧情、市场、帮派、低安玩法。
- 付费只卖外观、舒适度、功能位便利，以及严格受限的行具外形与功能扩展，不提升核心战斗预算。

## 3. 核心循环与日周期

### 3.1 宏观循环

```text
生产/采集 -> 制造/炼药 -> 区域市场/合同流通 -> 挂机/PvE/PvP 消耗
    ^                                                   |
    |___________________ 银两回收与材料回流 _____________|
```

### 3.2 玩家日周期

- 早上 5 分钟：看江湖日报 -> 收离线产出 -> 调整今日队列 -> 检查挂单与帮派简报。
- 午休 3 分钟：领日常、微调挂单、领取队列阶段成果。
- 晚上 30-90 分钟：副本、远征、护镖、劫镖、擂台、帮战、世界事件择一或组合。
- 睡前 5 分钟：设过夜队列、整理战利品、补保险/护命符、查看次日路线机会。

### 3.3 两类玩家的收益边界

| 维度 | 休闲/收菜玩家 | 肝/深度玩家 | 设计要求 |
| --- | --- | --- | --- |
| 日均在线 | 15-30 分钟 | 1-4 小时 | 两者都能推进核心账号成长 |
| 基础成长 | 1.0 | 1.35-1.6 | 深度玩家优势体现在效率和高端内容，不体现在剥夺休闲体验 |
| 剧情体验 | 100% | 100% | 不以高强度在线限制主线 |
| 市场参与 | 卖材料、买成品 | 套利、囤货、控线 | 两类玩家互相成为对方供需 |
| PvP 深度 | 可完全回避预约/危险内容 | 可深入路线战、据点战 | PvP 不应成为 PvE 的前置门槛 |

### 3.4 风险与收益梯度

首发推荐的“理论毛收益/小时”与“熟练玩家净收益/小时”口径如下：

| 区域 | 理论毛收益 | 熟练玩家净收益 | 不熟练玩家净收益 | 核心用途 |
| --- | --- | --- | --- | --- |
| 高安 | 1.0 | 1.0 | 1.0 | 保底成长、生活技能、低压经营 |
| 低安 | 1.25 | 1.15 | 1.0 | 轻风险高周转、护镖与密报前置 |
| 危险区 | 1.7 | 1.35 | 0.9 | 税权、资源腹地、高价值制造材料 |
| 伏龙外围事件期 | 2.1 | 1.55 | 0.8 | 远征、世界事件、势力冲突放大器 |

结论：危险区不是“无脑更赚”，而是“组织化、配装正确、路线判断正确才更赚”。

## 5. 地图、路线与势力规则

### 5.1 地图拓扑原则

- 地图不是连续平面，而是“节点江湖”。
- 每次移动都是从一个地区节点进入另一个地区节点。
- 真正的 PvP 接战只发生在节点落点，而不发生在中间路径表现层。
- 路线选择、假动作、堵口、反包围，是路线玩法核心。

### 5.2 节点类型规则

| 节点类型 | 主要功能 | 能否驻防 | 能否征税 | 典型叙事 |
| --- | --- | --- | --- | --- |
| 主城 | 拍卖、官署、声望、跨区物流 | 否 | 系统税 | 官府与商盟秩序中心 |
| 县镇 | 补给、日常、地方制造 | 否 | 系统税 | 人情往来与地方任务 |
| 渡口 | 水路切换、镖运、截镖热点 | 否 | 部分过路税 | “等你过河再动手” |
| 山关 | chokepoint、伏击、防守 | 否 | 否 | 视野差、堵口、反包围 |
| 矿镇 | 资源、工坊、据点争夺 | 是 | 资源税 | 地区经济命门 |
| 寨堡 | 玩家/NPC 驻地、仓站、修理 | 是 | 交易税、过路税 | 边疆秩序的形成点 |
| 古闸/遗址 | 远征、副本、主线异象 | 事件期 | 否 | 九闸旧案与玄灯线索 |

### 5.3 路线类型规则

| 路线类型 | 基础耗时系数 | 风险系数 | 运输效率 | 玩法用途 |
| --- | --- | --- | --- | --- |
| 官道 | 1.0 | 0.8 | 中 | 稳定跑商、日常运输 |
| 水道 | 0.9 | 1.1 | 高 | 轻货快运、截镖、绕行 |
| 栈道 | 1.2 | 1.3 | 低 | 小队伏击、山关教学 |
| 地脉密径 | 0.7 | 1.4 | 低 | 高阶绕后、奇遇、追逐反转 |
| 季节性险路 | 0.8 或 1.5 | 1.6 | 中 | 世界事件期间改变战线 |

### 5.4 安全等级规则

| 安全等级 | PvP 规则 | 死亡规则 | 收益系数 | 适用人群 |
| --- | --- | --- | --- | --- |
| 高安 | 禁止自由 PvP，仅限擂台、切磋、预约玩法 | 无装备损毁，仅耐久损耗 | 1.0 | 新手、休闲、制造、市场 |
| 低安 | 需主动开红，攻击方承担恶名与赏金惩罚 | 货物掉落、装备低概率损毁 | 1.25 | 护镖、劫镖、谨慎刷匪 |
| 危险区 | 自由 PvP，无系统法律惩罚 | 货物大量掉落、装备中概率损毁 | 1.7 | 帮派、重甲刷子、危险区后勤 |
| 伏龙外围事件态 | 自由 PvP，世界事件期间路线权放大 | 多槽位损毁判定 | 2.1 | 高端远征、事件争夺 |

### 5.5 首发路线玩法规则

- 玩家移动分三段处理：出发确认 -> 路线行进表现 -> 落点结算。
- 客户端表现可以展示“正在通过某条官道/水道”，但服务端只在出发与落点进行核心判定。
- 落点判定包括：
  - 是否进入安全/争议/危险节点。
  - 是否触发帮派驻防、悬赏警示、敌对包围圈、世界事件边界。
  - 是否解锁本地市场、补给、税收、修理、主线节点。

### 5.6 路线追逐与接战规则

PvP 追逐固定分为五步：发现 -> 试探 -> 截断 -> 反包围 -> 接战。

- 发现：通过哨探、挂单异常、节点驻防、通缉提示、系统事件暴露线索发现目标。
- 试探：双方在相邻节点间停留、试移、虚晃，判断真实线路。
- 截断：猎人提前占领渡口、山关、矿镇等 choke 节点。
- 反包围：目标通过帮派频道呼叫同盟，从另一条边切入。
- 接战：双方在同一节点落地后进入战斗结算。

系统不提供“敌人靠近弹窗”，只提供玩家驱动的情报系统和地图公共痕迹。

### 5.7 六大势力与系统职责

| 势力 | 系统职责 | 玩家感知 |
| --- | --- | --- |
| 晟朝漕运司 | 高安秩序、通行榜文、新手安全边界、主线官署任务 | 高安为什么稳定、低安为什么开始漏风 |
| 四通商盟 | 区域市场、合同、保险、价格公示、贸易任务 | 市场流通、商路价值、价差与信用 |
| 栖霞剑宗 | 擂台、榜单、武学试炼、正道声望、名器线 | 武侠名望与战斗荣誉 |
| 慈渡禅院 | 复活、护命符、伤后保护、新手救助、医术线 | “死亡不是退坑”的心理底座 |
| 青云帮盟 | 帮派、哨探、镖队、边州秩序、危险区自治样板 | 有组织就更安全、有江湖就有故事 |
| 玄灯教 | 深层远征、九闸旧案、伏龙事件、赛季冲突核心 | 高危内容与深层不安的来源 |

### 5.8 危险区自治与税收

首发可占领节点限定为 `朔砂外原` 的矿镇/寨堡类节点与 `伏龙外围前哨`。

可征税类型固定三类：

- 交易税：本节点市场、委托、拍卖、合同成交抽成。
- 过路税：节点作为交通 choke 点时，对运输行为抽成。
- 资源税：在本节点发生的采集、开采、加工行为抽成。

税率限制：

- 交易税：3% - 7%。
- 过路税：2% - 6%。
- 资源税：5% - 12%。

系统限制：

- 单节点税率 24 小时内只能改动一次。
- 税率每次改动幅度不超过 2 个百分点。
- 税率高于推荐阈值后，节点满意度下降，触发 NPC 流失、黑市绕路、敌对举报概率提升。
- 危险区主权存在维护费，不允许“占了就躺吃”。

节点维护费公式：

`每日维护费 = 500 x 节点级别 + 200 x 驻防级别 + 150 x 建筑数量`

### 5.9 地图系统研发视图

#### 核心状态流转

- `未发现` -> `已解锁可见` -> `可进入` -> `可驻留/交易/战斗` -> `事件态/战争态`。

#### 服务端判定点

- 出发时校验目标节点是否开放、路线是否可用、货物是否符合限制。
- 落点时判定安全等级、驻防、帮派控制、税率、通缉、世界事件。
- 战争态按节点和时间窗开启，不按客户端请求决定。

#### 客户端展示点

- 节点详情卡展示安全等级、资源、势力、税率、近 24 小时热度。
- 路线预览展示基础耗时、风险提示、是否有季节/事件加成。
- 接战节点需要显示最近战报与情报热度。

#### 可配置项

- 节点开放章节、路线耗时系数、资源标签、税率区间、事件开关。

#### 埋点指标

- 节点到达率、路线选择率、节点驻留时长、运输中断率、围堵成功率。

#### 异常分支

- 目标节点关闭：返还一半行程时间并转入最近可达节点。
- 路线事件切换：保留原行程，落点前重新判定事件态。
- 市场圈临时封锁：只影响新单，不影响已挂单结算。

## 6. 角色成长、技能与构筑

### 6.1 成长总原则

- 不使用传统职业锁死。
- 角色成长由技能熟练、领悟节点、装备、心法、行具和势力声望共同构成。
- 玩家展示位是“江湖身份”，不是“固定职业”。

### 6.2 核心战斗属性

| 属性 | 作用 |
| --- | --- |
| 气血 | 生存上限 |
| 内力 | 主动技能与心法驱动资源 |
| 外攻 | 刀剑拳等外功伤害 |
| 内攻 | 心法、暗器、特殊武学伤害 |
| 外防 | 对外攻减伤 |
| 内防 | 对内攻减伤 |
| 命中 | 降低目标闪避 |
| 闪避 | 躲避普通攻击和部分技能 |
| 暴击 | 造成高额伤害概率 |
| 韧性 | 降低被暴击收益 |
| 治疗效率 | 药品、医术、恢复技能收益 |
| 负重 | 决定可携带货物与装备重量 |

### 6.3 技能分组

#### 战斗技能

- 剑法
- 刀法
- 拳掌
- 暗器
- 内功

#### 生活技能

- 采药
- 采矿
- 伐木
- 锻造
- 炼药
- 烹饪
- 裁缝
- 奇门

#### 特殊技能

- 轻功：移动耗时、脱战能力、运输规避。
- 医术：战后恢复、队伍救护、死亡减损。
- 经商：税费减免、挂单效率、合同收益。
- 侦听：哨探报告质量、追踪效率、路线识别。

### 6.4 技能成长规则

每个技能由 `熟练度等级` 与 `领悟等级` 两层组成。

- 熟练度等级：1-100，主要来自使用该技能对应行为。
- 领悟等级：0-10，通过里程碑、残卷、名师传授、剧情节点获得。

推荐熟练度公式：

`升级所需熟练经验(n) = 20 + 8 x n^1.4`

推荐领悟规则：

- 每 10 级熟练度解锁 1 个领悟槽。
- 领悟点通过以下方式获取：
  - 主线章节奖励。
  - 远征残卷。
  - 师徒教学奖励。
  - 势力声望商店。

领悟节点类型：

- 战斗节点：伤害、减伤、暴击、回复。
- 生活节点：产量、品质、制造耗时、材料损耗。
- 特殊节点：跑商安全、税费减免、情报质量、脱战成功率。

### 6.5 构筑规则

`CharacterBuild` 固定由以下模块组成：

- 主武器 1 个。
- 副位 1 个，允许副武器或防具特装。
- 防具 4 个：头、身、腕、足。
- 饰品 2 个。
- 心法 2 个：主修、辅修。
- 丹药位 2 个：战斗丹与回复丹。
- 机关位 1 个。
- 行具位 1 个。

### 6.6 装备品质与来源

| 品质 | 主要来源 | 首发定位 |
| --- | --- | --- |
| 白 | 基础制造、低级匪巢 | 廉价流消耗品 |
| 绿 | 制造、低安掉落 | 新手毕业与低成本挂机 |
| 蓝 | 中级制造、副本 | 主流过渡装备 |
| 紫 | 高级制造、远征、副本首领 | 高端日常、帮战基础装 |
| 橙 | 极稀有远征、世界 Boss | 稀缺高价值资产 |
| 势力装 | 匪巢、远征、势力剧情 | 定制流派强化 |
| 遗物 | 命名精英、伏龙外围、赛季接口 | 首发最高端收藏与核心资产 |

### 6.7 行具系统

为统一跑商、运输、槽位与“底盘式损耗”体验，首发采用抽象的 `行具` 系统。行具在叙事上表现为马匹、驮架、轻舟、剑匣、机关架等，规则上作为角色的“移动与装载底盘”。

行具类型：

| 行具类型 | 特点 | 适合玩法 |
| --- | --- | --- |
| 轻行具 | 速度高、货仓小、脱战好 | 哨探、密信、跑图 |
| 镖运具 | 货仓大、防护中等 | 跑商、护镖 |
| 战旅具 | 战斗槽位高、负重高 | 危险区刷匪、帮战 |
| 机关具 | 功能槽多、采集与制造加成 | 生活流、后勤 |

行具字段：

- 基础速度
- 货仓容量
- 功能槽位
- 战斗预算
- 隐匿修正
- 维修成本

首发原则：付费行具只能增加 `功能槽`，不增加 `基础战斗预算` 与 `核心战斗槽`。即使商城销售行具，也必须保持“舒适度扩展”而不是“战力上限提升”。

### 6.8 成长系统研发视图

#### 核心状态流转

- `未掌握` -> `已学习` -> `熟练提升` -> `领悟解锁` -> `构筑固化/切换`。

#### 服务端判定点

- 技能经验结算、领悟点消耗、装备属性汇总、行具预算校验。
- 战斗前快照固化，战斗中不允许更换关键槽位。

#### 客户端展示点

- 构筑页需同时显示战斗收益、生活收益、负重、跑商风险。
- 技能树需清楚区分熟练成长和领悟节点。

#### 可配置项

- 技能经验曲线、领悟节点、装备品质倍率、行具类型与预算。

#### 埋点指标

- 技能使用率、构筑切换率、行具类型占比、不同品质装备的平均持有时长。

#### 异常分支

- 构筑超负重：无法出发运输与战斗。
- 关键装备过期或损毁：队列转为 `待补装` 状态。

## 7. 行动队列、挂机与离线结算

### 7.1 队列基础规则

- 免费玩家基础行动队列：4 槽。
- 单槽最短时长：15 分钟。
- 单槽最长计划时长：8 小时。
- 全队列最长规划视野：24 小时。
- 离线结算上限：24 小时。

首发行动类型：

- 采集
- 制造
- 炼药
- 烹饪
- 打坐
- 清剿匪巢
- 区域运输委托
- 修理/整理

### 7.2 队列状态

`planned` -> `running` -> `completed`

并允许以下异常态：

- `paused`：玩家手动暂停。
- `interrupted`：因战利品栏满、装备损毁、货仓不足、节点关闭而中断。
- `await_claim`：已完成但产出未领取。
- `await_manual_next`：匪巢已清完，等待玩家手动点“前往下一处”。

### 7.3 离线结算公式

统一公式：

`实际产出 = 基础效率 x 技能系数 x 设施系数 x 区域系数 x 行具系数 x 时长 x 完成系数`

其中：

- `基础效率`：由行为模板决定。
- `技能系数`：`1 + 0.02 x 相关技能熟练档位 + 0.03 x 相关领悟档位`。
- `设施系数`：有工坊/药炉/驻地支持时取 1.05 - 1.2。
- `区域系数`：高安 1.0，低安 1.1，危险区 1.25。
- `行具系数`：只对运输、采集、整理生效，取 0.95 - 1.15。
- `完成系数`：正常 1.0，中断按实际完成比例折算。

设计说明：

- 离线收益不打折，在线优势来自奇遇、命名精英、帮派情报响应、手动选择更优路线，而不是系统直接给在线玩家加产出。
- 这样既保住放置感，也保留晚间主动游玩的意义。

### 7.4 匪巢半挂机规则

- 一次匪巢清剿时长为 8-22 分钟。
- 清剿完成后不会自动跳到下一处。
- 玩家必须手动选择 `前往下一处`，否则角色停在本地休整，不受额外损失。
- 该停顿既是上班族操作点，也是猎人和哨探的信息窗口。

### 7.5 仓储、货仓与战利品规则

- 材料、制造产出进入仓库。
- 运输货物进入行具货仓。
- 匪巢与副本产物先进入战利品栏。

容量规则：

- 仓库基础格：80。
- 战利品栏基础格：20。
- 行具货仓由行具类型决定。

溢出规则：

- 仓库满：新产出进入 `堆场待取`，最多缓存 2 小时产量，超出后对应队列暂停。
- 战利品栏满：当前匪巢结算后，行动转入 `await_claim`，不再继续下一次清剿。
- 货仓超载：运输无法出发。

### 7.6 异常中断与恢复

以下情况会中断队列：

- 核心装备损毁导致战力不满足最低门槛。
- 行具损毁导致货仓不足。
- 战利品栏/仓库溢出。
- 目标节点因战争或剧情临时关闭。
- 玩家被击杀导致正在运输的货物结算。

恢复规则：

- 玩家处理异常并补足条件后，可一键从中断点恢复。
- 恢复不返还已损失时间，但保留未完成部分规划时长。

### 7.7 挂机行为的随机事件

首发在线随机事件：

- 命名精英现身。
- 山寨奇遇文本分支。
- 商会临时委托。
- 帮派求援广播。
- 路线传闻刷新。

离线期间不触发需要实时决策的事件，但保留基础掉落与密报触发。

### 7.8 队列系统研发视图

#### 核心状态流转

- `planned` -> `running` -> `await_claim` -> `claimed`。
- 异常支路：`running` -> `interrupted` -> `resumed`。
- 匪巢支路：`completed` -> `await_manual_next` -> `running(next)`。

#### 服务端判定点

- 每个槽位开始时固化参数。
- 每次结算时检查仓储、战力、节点状态、货仓状态。
- 匪巢清剿结束时生成战利品包和公共痕迹值。

#### 客户端展示点

- 队列总览、槽位剩余时长、异常原因、可一键恢复。
- 匪巢结算弹窗、前往下一处按钮、战利品栏满预警。

#### 可配置项

- 槽位数、最短/最长时长、仓库格数、战利品栏格数、堆场缓存时长。

#### 埋点指标

- 队列完成率、异常中断率、恢复率、战利品栏满频次、夜间离线时长。

#### 异常分支

- 玩家离线超 24 小时：只结算前 24 小时产出。
- 队列跨版本热更：旧槽位执行旧模板到结束，避免中途变参。

## 8. 经济、市场与制造

### 8.1 四层经济结构

| 层 | 内容 | 首发说明 |
| --- | --- | --- |
| 生产 | 采集、制造、任务、匪巢残骸、副本产物 | 基础材料必须可持续注入 |
| 积累 | 装备、心法、图纸、仓储、声望、行具 | 形成玩家资产与长期目标 |
| 流通 | 区域市场、合同、摆摊、运输、护镖 | 让供需在路上相遇 |
| 消费 | PvP 损毁、副本消耗、税、修理、突破、维护、保险 | 维持市场消耗与银两回收 |

### 8.2 货币与价值锚

首发货币不做复杂化，采用三层：

- 银两：全局主货币，系统主要注入口为赏金、任务、订单；主要排水口为税、维护、修理、突破、保险。
- 声望：势力成长与章节门槛，不可自由交易。
- 帮贡：帮派内循环货币，用于驻地、情报、驻防、内部福利，不可对外流通。

价值锚：

- 低级材料始终有 NPC 回收底价，防止完全归零。
- 高级材料价格不由系统定价，完全由玩家成交中位数形成参考值。
- 保险、维护、世界事件入场费用引用过去 7 天成交中位价作为估值。

### 8.3 区域市场结构

首发使用 4 个市场圈：

- `高安核心圈`：临渚中州 + 青蒲水网。
- `边渡水网圈`：落雁边渡 + 雾泽连汊。
- `山关群谷圈`：黑风群谷 + 断云岭道。
- `外原前哨圈`：朔砂外原 + 伏龙外围前哨。

规则：

- 挂单只在本市场圈可直接成交。
- 合同全服可见，但提货/交付必须到指定圈完成。
- 价差是跑商与护镖的根本动机。

### 8.4 价格形成与价差口径

区域目标成交价：

`区域参考价 = 最近 7 天成交中位价 x 区域稀缺系数 x 战争需求系数`

其中：

- `区域稀缺系数`：0.85 - 1.35，由本地资源产出与最近库存消耗决定。
- `战争需求系数`：1.0 - 1.25，由帮战、世界事件、节点维护需求决定。

跑商预估利润：

`预估利润 = 目标圈参考价 - 来源圈购入价 - 交易税 - 过路税 - 运输成本 - 预期损失成本`

### 8.5 银两排水口目标占比

| 排水口 | 目标占比 | 说明 |
| --- | --- | --- |
| 交易税 | 5%-8% | 主回收口，与市场成交直接挂钩 |
| 制造费/设施费 | 3%-5% | 稳定、低痛感 |
| 传送/运输费 | 2%-3% | 刺激路线选择 |
| 帮派维护费 | 2%-4% | 压制永久滚雪球 |
| 保险金 | 1%-3% | 风险对价 |
| 心法突破 | 可变 | 中后期回收口 |
| 修理费 | 1%-2% | 小额高频 |
| 仓库/摊位扩容 | 少量稳定 | 主要为便利回收 |

### 8.6 制造规则

制造公式：

`蓝图 + 主材槽 + 辅材槽 + 时间 + 技能/设施 -> 成品`

首发制造目标：

- 白、绿、蓝、紫品质必须可制造。
- 橙品质仅少量高端图纸可制造，依赖远征材料。
- 势力装与遗物不参与常规制造。

### 8.7 主材槽与材料变体

同一蓝图通过不同主材决定成品子类型。例如：

| 蓝图 | 主材 | 成品倾向 |
| --- | --- | --- |
| 长剑蓝图 | 寒铁 | 偏命中、破防 |
| 长剑蓝图 | 赤铜 | 偏暴击、灼伤 |
| 长剑蓝图 | 青钢 | 偏韧性、耐久 |
| 轻甲蓝图 | 雾丝布 | 偏闪避 |
| 轻甲蓝图 | 黑风皮 | 偏气血 |

规则要求：

- 每个蓝图至少 1 个主材槽支持 2-3 种替代材料。
- 同一成品类别至少依赖 2 种以上来源路径，例如 `采集 + 匪巢残骸` 或 `采集 + 远征掉落`。
- 防止单一据点完全卡死某类成品供给。

### 8.8 资源来源要求

首发关键资源要遵循以下原则：

- 核心基础材料：高安可稳定产出。
- 中级进阶材料：低安和副本共供给。
- 高级制造材料：危险区与远征主供给，但必须保留少量替代方案。
- 势力专属材料：绑定剧情、远征、势力商店，不直接来自纯挂机。

### 8.9 反垄断缓冲机制

当系统检测到以下任一条件时，启动反垄断缓冲：

- 同一联盟连续 7 天控制某类危险区核心资源供给超过 45%。
- 同一市场圈某关键材料 7 日均价高于全服中位价 180%。
- 同一联盟同时占领 3 个以上高税节点并维持 5 天以上。

缓冲措施：

- 非占领区的替代资源产量提升 10%-15%。
- 黑市合同刷新，向对手阵营提供破局任务与隐藏路线。
- 被垄断节点维护费递增 20%-40%。
- 系统事件优先在垄断区触发骚乱/匪患/NPC 罢市，逼迫其投入防守成本。

### 8.10 经济系统研发视图

#### 核心状态流转

- `生产` -> `库存` -> `挂单/运输` -> `成交/消耗` -> `银两回收/材料回流`。

#### 服务端判定点

- 成交税计算、参考价更新、保险估值、节点维护结算、反垄断检测。

#### 客户端展示点

- 市场挂单簿、历史价格图、区域稀缺提示、运输利润预估。

#### 可配置项

- 市场圈划分、税率、材料来源、蓝图主材槽、反垄断阈值。

#### 埋点指标

- 周银两净注入、单品成交量、跨圈运输频次、税收回收量、价格波动率。

#### 异常分支

- 低成交品类估值缺失：采用 NPC 回收价和相邻材料篮子估值。
- 价格被刷单污染：保险和系统成本估值使用剔除异常成交后的中位价。

## 9. 匪巢、掉落、远征与 PvE

### 9.1 匪巢档位

| 档位 | 主要区域 | 单次时长 | 推荐战力档 | 基础赏金 | 密报概率 |
| --- | --- | --- | --- | --- | --- |
| T1 | 临渚、青蒲 | 8-10 分钟 | 入门 | 80-120 两 | 0% |
| T2 | 落雁外围 | 10-12 分钟 | 低阶 | 120-180 两 | 2% |
| T3 | 黑风、雾泽 | 12-15 分钟 | 进阶 | 180-260 两 | 5% |
| T4 | 断云、低安深处 | 14-18 分钟 | 中阶 | 260-380 两 | 7% |
| T5 | 朔砂外原 | 18-20 分钟 | 高阶 | 420-650 两 | 8% |
| T6 | 伏龙外围 | 20-22 分钟 | 高阶以上 | 700-1000 两 | 10% |

### 9.2 匪巢基础流程

`选择区域 -> 进入匪巢 -> 自动战斗 -> 结算银两/残骸/装备/密报 -> 玩家手动选择下一处`

设计意图：

- 让完全挂机有自然停顿，不形成无限永动。
- 给摸鱼玩家稳定节奏，每 10-20 分钟点一次即可。
- 给路线追逐与帮派预警提供操作窗口。

### 9.3 掉落结构

每次清剿固定分为四档产出：

- 赏金银两：必出，系统注入银两。
- 残骸/战利品：必出，进入制造链。
- 低级装备：中概率，直接使用、分解或上架。
- 稀有产出：势力装、遗物、密报，小概率。

推荐掉落口径：

- 白绿装备基础掉落率：35%-55%。
- 蓝装掉落率：8%-15%。
- 势力装掉落率：0.6%-2.0%。
- 遗物掉落率：0.05%-0.2%。

### 9.4 掉落结算公式

`银两赏金 = 模板基础值 x 区域收益系数 x 清剿效率系数`

`残骸数量 = 模板残骸组 x (1 + 生活技能附加) x 随机波动`

`装备品质权重 = 模板权重 x 命名精英加成 x 世界事件加成`

其中：

- `清剿效率系数` 推荐区间 0.9 - 1.15，用于奖励正确构筑但不放大过头。
- `随机波动` 推荐区间 0.85 - 1.15。

### 9.5 命名精英与公共痕迹

命名精英规则：

- T3 以上匪巢才有概率刷出命名精英。
- 基础概率：T3 为 8%，T4 为 10%，T5 为 12%，T6 为 15%。
- 命名精英显著提高势力装、遗物、密报概率。

公共痕迹规则：

- 每清完一次匪巢，生成 1 点 `行踪痕迹值`。
- 同区域连续清剿时痕迹叠加，最大 5 点。
- 每点痕迹持续 12 分钟，可被哨探或猎人通过区域观察发现。
- 切换区域或 15 分钟不继续清剿后痕迹自然衰减。

该机制代替系统预警，为“猎人能抓到人、帮派能保住人”提供共同信息底座。

### 9.6 远征与密报规则

密报获取：

- 来自 T3-T6 匪巢结算。
- 可交易。
- 有效期 24 小时。

远征模板分三档：

| 远征档位 | 队伍规模 | 主要来源 | 主要奖励 |
| --- | --- | --- | --- |
| 初级密报 | 1-3 人 | 黑风、雾泽 | 蓝紫装、势力材料、少量图纸 |
| 进阶密报 | 3-5 人 | 断云、朔砂 | 紫装、势力装、高级图纸 |
| 深层密报 | 5 人 | 朔砂、伏龙外围 | 橙装材料、遗物概率、第三幕终局前置 |

远征与常规副本区别：

- 远征靠运气或交易获得，不可稳定刷新。
- 常规副本按周期开放，承担稳定供给职责。
- 远征提供“开彩票”情绪与高端资产供给。

### 9.7 常规 PvE 内容切片

首发 PvE 内容建议：

- 单人剧情副本：3 套。
- 3 人队副本：3 套。
- 5 人高端副本：2 套。
- 世界 Boss：2 个常驻轮换 + 1 个第三幕事件 Boss。
- 日常任务：6 类，任选 3 类完成日活。
- 周常任务：4 类，奖励高品质材料与声望。

### 9.8 PvE 与 PvP 的关系

- PvE 产出为经济底座。
- PvP 是高价值装备和行具的主要消耗出口。
- 远征是两者之间的桥：玩家可以靠挂机刷出密报，也可以靠市场购买密报再主动参与高端内容。

### 9.9 PvE 系统研发视图

#### 核心状态流转

- `生成模板` -> `进入实例` -> `自动战斗` -> `掉落结算` -> `等待下一处/返回`。
- 远征支路：`获得密报` -> `交易或自用` -> `组队进入` -> `限时结算`。

#### 服务端判定点

- 匪巢模板抽取、掉落池判定、命名精英判定、痕迹值生成、密报生成。

#### 客户端展示点

- 清剿结算面板、密报信笺展示、远征倒计时、命名精英特写。

#### 可配置项

- 匪巢档位、赏金、掉落率、痕迹持续时间、密报概率、远征时限。

#### 埋点指标

- 匪巢完成时长、密报产出率、远征参与率、命名精英击杀率。

#### 异常分支

- 密报过期：自动转为残页兑换材料。
- 队伍人数不足：允许补位倒计时，不直接吞掉密报。

## 10. PvP、死亡与风险倾斜

### 10.1 PvP 玩法分层

- 擂台：公平匹配、纯荣誉、仅耐久损耗。
- 劫镖/护镖：围绕运输行为的路线对抗。
- 低安开红：有限制、有惩罚、有通缉。
- 危险区自由 PvP：消耗与政治核心出口。
- 帮战/据点战：预约时段、大规模政治玩法。

### 10.2 低安与危险区的开红规则

低安：

- 攻击前需主动开红。
- 开红持续 30 分钟。
- 期间进入高安会被守卫驱赶，无法使用部分主城功能。
- 开红成功击杀将增加恶名并进入通缉池。

危险区：

- 不需要开红。
- 无法律惩罚，但会触发帮派恩怨、地区仇恨、悬赏与反击。

### 10.3 死亡结算顺序

PvP 死亡按以下顺序结算：

1. 结算当前运输货物与战利品。
2. 掉落非绑定货物。
3. 对高价值装备位进行损毁判定。
4. 计算保险赔付与护命符保护。
5. 扣除耐久并生成残片。
6. 玩家在最近安全复活点复活并获得保护期。

### 10.4 货物掉落规则

| 区域 | 货物掉落率 | 战利品掉落率 |
| --- | --- | --- |
| 低安 | 40% | 50% |
| 危险区 | 70% | 80% |
| 伏龙外围事件态 | 85% | 100% |

说明：

- 货物是真正的海盗收益主体。
- 装备主要以损毁回收而不是整件爆出，避免一次击杀让普通玩家倾家荡产。

### 10.5 装备损毁规则

参与损毁抽取的核心槽位：主武器、副位、防具主体、心法、行具。

抽取数量：

- 低安：随机抽取 1 个槽位。
- 危险区：随机抽取 2 个槽位。
- 伏龙外围事件态：随机抽取 3 个槽位。

损毁概率公式：

`损毁概率 = 区域基础值 x 品质系数 x (1 - 保险减免) x (1 - 护命减免)`

区域基础值：

- 低安：6%
- 危险区：12%
- 伏龙外围事件态：18%

品质系数：

- 白：1.0
- 绿：1.1
- 蓝：1.25
- 紫：1.5
- 橙：1.8
- 势力装：1.7
- 遗物：2.0

若未损毁，则装备耐久降低 40%。

### 10.6 保险与护命符

保险分两档：

| 保险类型 | 保费 | 损毁减免 | 销毁后赔付 |
| --- | --- | --- | --- |
| 标准保 | 参考值 8% | 30% | 参考值 50% |
| 全保 | 参考值 15% | 50% | 参考值 65% |

规则：

- 参考值来自过去 7 天成交中位价，避免玩家自己抬价骗保。
- 保险只赔银两，不返还原物。
- 货物与密报不享受保险。

护命符规则：

- 每张护命符绑定 1 个装备槽位。
- 触发后本次损毁概率视为 0，但仍扣 20% 耐久。
- 同一槽位 24 小时内只能触发 1 次护命符。
- 护命符可由炼药/奇门联合制造，也可少量通过活动获取。

### 10.7 重复击杀保护

为防骚扰和刷收益，采用同攻击源收益衰减：

| 同一攻击者在 120 分钟内击杀同一受害者次数 | 掉落收益系数 | 赏金收益系数 |
| --- | --- | --- |
| 第 1 次 | 100% | 100% |
| 第 2 次 | 40% | 40% |
| 第 3 次及以上 | 0% | 0% |

附加保护：

- 受害者获得 15 分钟 `伤退庇护`，在同节点不可再次被同攻击源锁定。
- 若处于新手保护状态，首次被击杀后系统强制将其引导回高安恢复点。

### 10.8 新手保护

满足任一条件视为新手：

- 账号创建未满 72 小时。
- 主线未完成第四章。

保护效果：

- 高安和低安无装备损毁，只掉部分货物。
- 危险区不可携带高价值违禁类任务货物。
- 每日赠送 1 次基础护命。

### 10.9 廉价永动流与重甲速刷流

| 构筑 | 目标 | 预期收益 | 预期单次损失 | 操作要求 |
| --- | --- | --- | --- | --- |
| 廉价永动流 | 低成本稳定挂机 | 中低 | 低 | 低，可真 AFK |
| 重甲速刷流 | 高效率危险区清剿 | 高 | 高 | 高，需要情报、盯屏、撤离预案 |

设计要求：

- 廉价流要真实可行，而不是新手陷阱。
- 重甲流收益至少为廉价流的 2.5 倍，但单次死亡成本至少为其 8-15 倍。
- 猎人会自动优先筛选高价值目标，从而让廉价流成为自带安全策略的玩法。

### 10.10 护镖、劫镖与擂台

护镖/劫镖：

- 由 `TransportOrder` 驱动。
- 订单分普通货、急件、密信、军需四类。
- 风险越高、路线越长、时限越短，报酬越高。

擂台：

- 使用固定耐久损耗，不触发掉落与损毁。
- 奖励为声望、榜单积分、赛季称号、少量紫装材料。
- 是“武侠荣誉层”的主入口，不承担经济大额消耗职责。

### 10.11 PvP 系统研发视图

#### 核心状态流转

- `观察` -> `开红/锁定` -> `接战` -> `掉落与损毁结算` -> `通缉/保护期`。

#### 服务端判定点

- 开红合法性、攻击源归属、掉落计算、损毁判定、保险赔付、重复击杀收益限制。

#### 客户端展示点

- 开红提示、货物风险提示、击杀结算、通缉标记、保护期倒计时。

#### 可配置项

- 各区掉落率、损毁率、保护时长、保险费率、重复击杀衰减参数。

#### 埋点指标

- 区域击杀数、低安开红率、危险区净利润、护命符触发率、重复击杀比例。

#### 异常分支

- 跨节点追击中目标离线：落点后按离线防御规则处理，不允许强制原地判死。
- 多方混战归属争议：以最近有效伤害与控制占比判定主要击杀方。

## 11. 帮派、社交、情报与政治经济

### 11.1 帮派定位

帮派不是加 buff 容器，而是危险区安全感、情报效率、物流能力、据点政治和江湖恩怨的组织形式。

### 11.2 帮派岗位

| 岗位 | 权限/作用 |
| --- | --- |
| 帮主 | 政策、外交、据点任命、税率决策 |
| 副帮 | 代行管理、战时组织 |
| 战团长 | 帮战、护镖、截击指挥 |
| 哨探 | 地图巡视、情报上报、路线侦查 |
| 后勤 | 制造、仓储、补给分发、保险统筹 |
| 镖头 | 运输订单分配、护镖集结 |
| 外交 | 联盟、停战、仇敌、赏金协作 |

### 11.3 帮派情报频道

`GuildIntelReport` 为帮派的核心安全工具。

情报消息类型：

- 陌生战团出现。
- 某节点红名聚集。
- 某节点价格异常、疑似调货。
- 发现命名精英/世界 Boss。
- 发现敌方镖队或补给线。

情报规则：

- 一键上报当前节点与威胁类型。
- 每条情报默认持续 10 分钟。
- 情报有置信度，来源于上报者的侦听技能、最近误报率、是否有第二来源复核。
- 地图可视化只对帮派与同盟开放，不对全服广播。

### 11.4 帮派驻地与维护

- 帮派可在高安拥有非战斗型总舵，在危险区占领可税收节点。
- 驻地设施包括仓站、修理坊、瞭望塔、药房、合同栏。
- 危险区设施都消耗维护费和材料，不存在永久免费建设。

### 11.5 恩怨、师徒与市场社交

- 恩怨：被劫镖、被通缉、帮战历史自动生成恩怨记录，可展示在帮派与个人档案。
- 师徒：老玩家带新玩家完成首次副本、首次远征、首次成功跑商后，双方获得声望与少量功能奖励。
- 市场社交：装备和成品显示制造者，可关注“常去的老店”，形成商誉体系。

### 11.6 江湖日报

每日首次登录生成个人化 `江湖日报`，基础结构：

- 天下要闻：帮战、主线事件、世界 Boss、节点易主。
- 市价异动：本市场圈 3 类重点商品涨跌。
- 通缉榜：当日最活跃红名与赏金更新。
- 帮派简报：帮派税收、维护、情报热点、成员战报。
- 个人总结：离线产出、挂单成交、技能成长、图鉴进度。

日报刷新规则：

- 服务端每日 05:00 汇总世界版。
- 玩家首次登录时拼装个人版。
- 同日内支持“增刊”模块，用于重大帮战或事件快讯。

### 11.7 榜单体系

首发榜单：

- 武林榜：擂台与高端 PvE 综合荣誉。
- 富甲榜：银两与市场成交额综合。
- 铸匠榜：制造评分与成交口碑。
- 通缉榜：恶名与悬赏值。
- 帮会战绩榜：据点、护镖、劫镖、战争胜率综合。

### 11.8 帮派与社交系统研发视图

#### 核心状态流转

- `加入帮派` -> `承担岗位` -> `上报情报/参与物流/参加战事` -> `形成恩怨与战报`。

#### 服务端判定点

- 情报过期与置信度计算、帮派权限校验、驻地维护结算、榜单结算、日报生成。

#### 客户端展示点

- 帮派地图热区、情报频道、日报页面、制造者名片、恩怨史页面。

#### 可配置项

- 情报持续时间、榜单周期、驻地设施效果、帮派岗位权限。

#### 埋点指标

- 帮派加入率、情报上报率、护镖参与率、日报打开率、好友/师徒转化率。

#### 异常分支

- 情报刷屏：同类情报聚合为单条热点。
- 帮派解散：税收与设施进入冻结期，由系统托管 48 小时。

## 12. 叙事挂钩与章节推进

### 12.1 叙事接玩法原则

- 每章必须解锁至少 1 个核心系统或区域功能。
- 章节剧情必须解释系统出现的原因，而不是系统先出现、故事再补说明。
- 日报、势力任务、世界事件要承接主线推进后的余波。

### 12.2 前三幕章节与系统解锁表

| 章节 | 区域 | 叙事重点 | 系统解锁 |
| --- | --- | --- | --- |
| 第一章 `临渚初入` | 临渚中州 | 入世、漕运司秩序、江湖日报首见 | 基础行动队列、采集、主城市场、基础装备 |
| 第二章 `炉火青蒲` | 青蒲水网 | 工坊、人情、百工重开 | 锻造、炼药、图鉴、制造蓝图、仓库 |
| 第三章 `夜渡青蒲` | 青蒲 -> 落雁 | 第一次感知边州风险 | 运输订单、战利品栏、低安提示、日报个人页 |
| 第四章 `落雁封渡` | 落雁边渡 | 榜文与黑线并存 | 护镖/劫镖、低安开红、通缉系统 |
| 第五章 `黑风榜起` | 黑风群谷 | 山寨势力、旧榜重开 | T3 匪巢、命名精英、帮派招募 |
| 第六章 `雾中来客` | 雾泽连汊 | 密信、走私、错路 | 合同交易、情报系统、初级密报远征 |
| 第七章 `断云借道` | 断云岭道 | 关口与路权 | 山关 chokepoint 教学、哨探岗位、路线争夺 |
| 第八章 `外原立旗` | 朔砂外原 | 边疆自治、税收试行 | 危险区自由 PvP、据点税收、节点维护 |

### 12.5 叙事系统研发视图

#### 核心状态流转

- `章节触发` -> `节点开放/系统解锁` -> `日报播报` -> `区域余波任务`。

#### 服务端判定点

- 章节完成、系统解锁、NPC 状态切换、世界事件播报队列。

#### 客户端展示点

- 主线章节页、势力关系页、日报正文、节点剧情入口。

#### 可配置项

- 章节门槛、区域开放开关、日报模板、人物登场条件。

#### 埋点指标

- 章节完成率、章节后系统使用率、日报剧情点击率、人物支线开启率。

#### 异常分支

- 跳章风险：必须以系统解锁条件校验章节，不允许仅靠战力穿透。
- 热更剧情文案：与节点开放状态解耦，避免文案先于功能上线。

## 13. 商业化与公平边界

### 13.1 三条收入线

| 收入线 | 内容 | 边界 |
| --- | --- | --- |
| 时装/外观 | 时装、头像框、挂饰、行具皮肤 | 不加属性 |
| 舒适度道具 | 队列管理、一键收菜、仓储便利、保险代券、传送便利 | 不提高核心产出上限 |
| 行具售卖 | 珍藏行具外形与功能槽扩展 | 不增加核心战斗预算，不免损毁 |

### 13.2 舒适度道具规则

- 可售卖额外的 `队列管理权限`，例如批量编辑、模板保存、远程收菜。
- 不售卖额外离线结算时长，24 小时离线上限对所有玩家相同。
- 不售卖额外基础产能，只卖操作便利。

### 13.3 行具商业化优化方案

在保留原方案“商城可以直接售卖行具”的同时，采用以下限制：

- 付费行具仅多 1 个功能槽，不多战斗槽。
- 功能槽只能装载运输、整理、采集、外观类组件。
- 付费行具仍可损毁、仍需维修、仍受保险与护命规则约束。
- 擂台和部分公平玩法中统一折算为标准行具模板。

### 13.4 活动与运营节奏

- 每周一次帮派焦点活动：边州护镖周、矿镇争夺周、密报增刊周三选一。
- 每月一次世界事件：外原骚动、伏龙异象、商盟黑契轮换。
- 大版本再进入第四幕与沉灯海隙。

## 14. 埋点、平衡指标与验收

### 14.1 核心埋点

- 队列相关：日收菜次数、队列完成率、离线时长、异常中断率。
- 经济相关：银两注入/回收、关键材料流速、市场成交额、价格波动率。
- PvE 相关：匪巢完成率、密报触发率、远征完成率、命名精英击杀率。
- PvP 相关：低安开红率、危险区击杀数、货物掉落值、装备损毁值。
- 社交相关：帮派加入率、情报上报率、护镖参战率、日报打开率。
- 叙事相关：章节推进率、日报剧情点击率、人物支线开启率。

### 14.2 周平衡看板指标

| 指标 | 目标区间 | 预警含义 |
| --- | --- | --- |
| 周银两净注入 | +1.5% 至 +3.5% | 过高代表通胀，过低代表日常痛感过强 |
| 高价值装备日损毁/新增 | 0.28 至 0.35 | 过低市场停滞，过高玩家挫败 |
| 低安货运成功率 | 55% 至 70% | 过低说明猎杀过强，过高说明风险不足 |
| 危险区平均净收益比高安 | 1.25 至 1.45 | 过高会逼人人下危险区，过低会无人参与 |
| 帮派情报覆盖率 | 不低于 45% | 过低说明帮派玩法形同虚设 |
| 新手第四章前流失率 | 不高于 35% | 过高说明低安教育或痛感过重 |

### 14.3 八类玩家场景验收

| 场景 | 验收步骤 | 预期结果 |
| --- | --- | --- |
| 纯休闲收菜 | 连续 3 天每日 15 分钟登录 | 主线可推进、仓储可控、市场可参与 |
| 独狼生产 | 不入帮，专注高安制造和低安短线 | 可稳定盈利，不被强制推入 PvP |
| 帮派后勤 | 加入帮派，负责药品与维修供给 | 帮贡与银两都有正反馈 |
| 跑商护镖 | 承接跨圈订单并组织护送 | 价差、风险、报酬形成正循环 |
| 猎人截杀 | 在低安/危险区追击运输目标 | 能通过情报和路线判断抓到高价值目标 |
| 重甲速刷 | 组装高价值构筑刷外原 T5 匪巢 | 收益显著更高，但被抓损失也显著 |
| 远征小队 | 购买或自刷密报后组队远征 | 密报有价值，远征产出与难度匹配 |
| 市场操盘 | 跨圈囤货、观察世界事件涨跌 | 能从信息和时机中获利，而非纯刷钱 |

### 14.4 边界与反滥用验收

- 无限挂机：不能通过脚本或跳槽位实现无限自动跳匪巢。
- 仓储爆满：系统必须暂停而不是吞产出。
- 密报囤积：24 小时到期并转残页，不允许无限囤票。
- 重复蹲杀：同源收益衰减必须生效。
- 低价值号骗保：保险估值需剔除异常成交。
- 逃脱道具滥用：战斗中、控制中、世界事件封锁期必须有限制。
- 地区垄断失控：反垄断阈值触发后应有体感明显的缓冲动作。
- 新手连续受害：新手保护需覆盖第四章前的低安接触阶段。

### 14.5 交付验收标准

- 项目经理可按本文件拆系统需求、版本里程碑、模块负责人。
- 系统策划可按实体与配置簇建立首版数据表。
- 数值策划可直接根据本文公式与目标区间搭建首版数值。
- 客户端可根据模块研发视图拆页面与状态展示。
- 服务端可根据判定点建立结算、税收、掉落、通缉、情报和日报服务。

## 15. 默认值与后续扩展接口

### 15.1 首发默认值

- 只产出 1 份主机制文档，不额外拆分为子文档。
- 行具作为统一底盘系统进入首发，但以武侠包装呈现，不直接出现“船体”用词。
- 伏龙渊域只开放外围与前哨，不开放深层主权区。
- 第四幕、沉灯海隙、完整赛季战争只留字段和剧情接口。

### 15.2 后续版本可扩展方向

- 深渊级安全区规则与完整赛季积分。
- 沉灯海隙的海路贸易与异域材料。
- 更深层的遗物修复与名器收藏链。
- 跨端移动端简化操作层，但不改变核心经济口径。

### 15.3 本文档的研发优先级建议

P0 必须定死：

- 市场圈与地图圈层。
- 队列与离线结算规则。
- 匪巢半挂机与手动下一处。
- 低安/危险区死亡与损毁结算。
- 材料多来源与税收结构。

P1 必须跑通：

- 高安 -> 低安 -> 外原的完整成长闭环。
- 江湖日报、帮派情报、护镖/劫镖、密报远征。

P2 再扩展：

- 伏龙外围事件、第三幕高潮、反垄断动态事件。

至此，首发 MVP 已能实现“挂机 x 武侠叙事 x 玩家经济 x 路线 PvP”的核心承诺，并给后续赛季与资料片留下自然接口。
