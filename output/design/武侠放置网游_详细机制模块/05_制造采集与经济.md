# 05 制造采集与经济

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 建立首发玩家经济的底层供给，让材料、成品、税费、修理和保险形成闭环。
- 明确采集产区、蓝图配方、主材槽差异、区域装备家族和设施升级，使生产玩法既有深度也便于落地。
- 给数值策划和系统策划直接可用的“24 个通用配方 + 24 个区域家族配方 + 来源矩阵 + 成本估值模板”。

## 2. 解锁与入口

| 功能 | 解锁章节 | 入口 | 说明 |
| --- | --- | --- | --- |
| 采药/基础制造 | 第二章 `炉火青蒲` | 工坊页 | 新手可完成第一个制造闭环 |
| 采矿/伐木 | 第二章 `炉火青蒲` | 地图节点/工坊页 | 高安基础采集稳定开放 |

| 危险区高级材料 | 第八章 `外原立旗` | 朔砂外原/伏龙外围 | 高级制造主供给 |

## 3. 核心循环

`采集/清剿获得材料 -> 进入仓储 -> 选择蓝图与主材 -> 支付时间与设施成本 -> 获得成品 -> 自用或上架市场 -> 通过税费、修理、保险、维护费回收银两`

经济设计核心：

- 基础材料稳定注入，避免底层完全断货。
- 中高端材料主要来自低安、危险区和远征，形成价差和运输动机。
- 高价值成品必须有明确的损耗去处，不能只堆库存。

## 5. 公式与数值

### 5.1 制造公式

`成品 = 蓝图 x 主材槽系数 x 辅材完整度 x 技能系数 x 设施系数 x 区域家族系数 x 路线偏向修正 x 品质波动`

- `主材槽系数`：定义词缀倾向，不直接把数值拉爆
- `品质波动`：建议区间 `0.95 - 1.05`
- `卓品率`：基础 `3%`，锻造/炼药/裁缝/奇门相关等级每 10 级 +`0.6%`
- `区域家族系数`：仅放大功能倾向，不直接制造不可追赶的纯数值差
- `路线偏向修正`：定义装备或道具更适合的节点类型和路线场景

### 5.2 成本估值模板

`理论成本 = 材料市场中位价总和 + 设施费 + 时间折算费 + 风险运费`

`建议售价 = 理论成本 x 1.12 - 1.35`

### 5.3 反垄断缓冲

- 同一联盟连续 7 天控制高端资源超过 45%：替代产区产量 +10%
- 同一材料 7 日均价高于全服中位价 180%：刷新黑市合同
- 同一联盟占 3 个以上高税节点并维持 5 天：维护费 +20%-40%
