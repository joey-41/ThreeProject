# 09 帮派社交与政治经济

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 让帮派成为安全感、情报效率、物流能力和危险区长期目标的真正载体，而不是单纯加成容器。
- 明确岗位权限、情报结构、置信度、驻地建筑、税率、补给线和据点占领流程。
- 给社交与政治玩法可直接落到权限表、据点表、战报表、活动表和物流表的配置口径。

| 功能 | 解锁章节 | 入口 | 说明 |

## 3. 核心循环

`加入帮派 -> 承担岗位 -> 上报情报、参与物流、组织护镖或据点战 -> 获得帮贡、税收收益与战报 -> 强化驻地和协同能力 -> 形成更稳定的安全区和更激烈的危险区争夺`

帮派玩法的底层价值：

- 对休闲玩家：提供情报、护送、物资、被保护的安全感。
- 对深度玩家：提供税权、驻地、据点、补给线和敌对关系的长期目标。
- 对市场：提供军需需求、维护消耗和跨圈物流能力。

## 4. 条目库

### 4.1 帮派岗位权限

| 岗位 | 核心权限 | 日常职责 |
| --- | --- | --- |
| 帮主 | 任命岗位、定税率、宣战、同盟、拆建驻地 | 战略方向与重大决策 |
| 副帮 | 代行管理、审批入帮、开启活动 | 承担日常组织 |
| 战团长 | 发起护镖、据点战集结、战报标记 | 组织战斗玩家 |
| 哨探 | 上报高权重情报、标红节点、发追踪请求 | 巡路、观测、提前预警 |
| 后勤 | 仓站管理、维修分发、军需准备 | 保障材料与药品供应 |
| 镖头 | 分配运输订单、指定护送队 | 维持商路与跑商秩序 |
| 外交 | 修改关系标签、登记停战与仇敌 | 联盟与恩怨处理 |

### 4.2 情报消息结构

| 字段 | 说明 |
| --- | --- |
| `report_id` | 情报唯一 ID |
| `node_id` | 所在节点 |
| `route_band` | 所在路线带 |
| `threat_type` | 红名、陌生战团、价格异常、命名精英、Boss、镖队 |
| `confidence` | 置信度 0-100 |
| `reporter_id` | 上报者 |
| `expire_at` | 过期时间 |
| `supporting_reports` | 复核来源数量 |
| `copy_short` | 地图短文案 |

情报类型：

| 类型 | 用途 |
| --- | --- |
| 红名聚集 | 预警运输和刷点风险 |
| 陌生战团出现 | 帮战或截镖预警 |
| 价格异常 | 物流和囤货判断 |
| 命名精英出现 | 集结清剿或盯防猎人 |
| Boss/事件 | 帮派活动调度 |
| 友方镖队出发 | 护送协同 |

### 4.3 置信度算法口径

`置信度 = 基础 45 + 侦听等级 x 0.6 + 复核数 x 12 + 节点情报权重 - 最近误报惩罚 - 目标隐匿修正`

建议边界：

- 80 以上：红色高可信情报
- 60-79：黄色待确认情报
- 59 以下：灰色线索，仅建议关注

### 4.4 危险区六类组织目标

| 目标类型 | 说明 | 典型节点 |
| --- | --- | --- |
| 主税点 | 提供交易税、资源税与政治正当性 | 朔砂矿市 |
| 前哨点 | 集结、复整、宣战和战报中心 | 伏龙前哨 |
| 仓站点 | 共享库存与军需分发 | 烟窟旧堡 |
| 修理点 | 装备、行具与重伤修复 | 风蚀马场 |
| 观测点 | 世界事件预警、敌情确认 | 渊火观测台 |
| 补给线节点 | 多跳军需、断补给与反断补给核心 | 白骨盐道 |

### 4.5 驻地建筑

| 建筑 | 作用 | 建造成本 | 维护费 |
| --- | --- | --- | --- |
| 仓站 | 增加帮派共享库存和临时提货点 | 木材 40、铁料 20、银两 600 | 80/日 |
| 修理坊 | 成员修理费 -10% | 铁料 30、皮革 16、银两 520 | 70/日 |
| 瞭望塔 | 情报持续时间 +2 分钟 | 木材 28、风裂晶 8、银两 560 | 90/日 |
| 药房 | 帮派成员炼药耗材 -5% | 草药 36、药渣 18、银两 480 | 65/日 |
| 合同栏 | 帮派内部委托、军需派发 | 木材 24、纸张 20、银两 430 | 55/日 |
| 驿厩 | 镖队出发耗时 -6% | 木材 35、皮革 22、银两 580 | 75/日 |
| 关防架 | 关口驻防值提高 | 木材 30、黑砂矿 10、银两 620 | 95/日 |
| 观测盘 | 事件预警时间 +1 跳 | 风裂晶 12、薄银片 8、银两 640 | 100/日 |

### 4.6 推荐税率档

| 税种 | 保守档 | 平衡档 | 掠夺档 | 建议说明 |
| --- | --- | --- | --- | --- |
| 交易税 | 3%-4% | 5% | 6%-7% | 常用建议不高于 5% |
| 过路税 | 2%-3% | 4% | 5%-6% | 过高会逼护镖改线 |
| 资源税 | 5%-6% | 7%-8% | 9%-12% | 高税率仅建议短期战时使用 |

### 4.7 帮贡、恩怨、师徒与商誉

| 机制 | 规则 |
| --- | --- |
| 帮贡 | 护镖、据点维护、战报上榜、捐物资均可获得，用于内部福利与驻地建设 |
| 恩怨 | 被劫镖、被通缉、帮战交火会记录在案，可展示在帮派和个人档案 |
| 师徒 | 首次副本、首次远征、首次成功跑商带徒完成后，双方得声望与领悟点材料 |
| 商誉 | 装备与药品显示制造者，可收藏常去商户；重复成交提升“熟客推荐” |

### 4.8 据点占领流程

1. 帮主或副帮在开放时窗发起争夺  
2. 战团长布置前线驻点与补给  
3. 哨探上报敌对路线和补给线  
4. 成员在目标节点累计控制进度  
5. 占领成功后进入 24 小时保护建设期  
6. 保护期后开始正常征税并支付维护费  

## 5. 公式与数值

### 5.1 据点维护费

`每日维护费 = 500 x 节点级别 + 200 x 驻防级别 + 150 x 建筑数量`

### 5.2 帮派情报有效期

- 默认持续 10 分钟
- 有瞭望塔时提高到 12 分钟
- 高可信双人复核情报可持续 14 分钟

### 5.3 帮贡发放建议

| 行为 | 帮贡 |
| --- | --- |
| 上报高可信情报 1 条 | 8 |
| 完成普通护镖 1 单 | 12 |
| 完成军需护送 1 单 | 24 |
| 捐入指定军需材料 1 次 | 10-30 |
| 参与据点战且有效贡献达标 | 25 |
