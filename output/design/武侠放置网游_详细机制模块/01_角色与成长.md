# 01 角色与成长

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 定义首发角色成长的完整骨架，使玩家通过技能、心法、装备、领悟和行具形成不同江湖身份。
- 保证“不锁职业但有明显流派差异”，让 PvE、PvP、跑商、制造都能用同一套成长框架解释。
- 把成长系统细化到可直接建 `技能条目表`、`心法条目表`、`构筑快照表` 和 `属性成长表` 的粒度。

## 3. 核心循环

`学习技能 -> 使用行为积累熟练 -> 解锁领悟槽 -> 配置心法/装备/行具 -> 形成构筑 -> 在战斗/生产/运输中验证收益 -> 回流材料与经验 -> 再次迭代构筑`

首发设计强调三点：

- 熟练体现“你做过多少次”，领悟体现“你学到了什么窍门”。
- 心法与行具是构筑分歧的放大器，而不是独立职业。
- 任何构筑都必须同时看到战斗收益、生活收益、负重和风险表现。

| 青蒲药师 | 炼药 + 医术 + 奇门 + 机关具 | 帮派后勤 | 资源稳定、团队价值高 | 单刷效率一般 |

## 5. 公式与数值

### 5.1 熟练与领悟

- 熟练等级：1-100
- 领悟等级：0-10
- 熟练经验公式：`20 + 8 x n^1.4`
- 生活技能使用经验：每次行为基础 4-12 点
- 战斗技能使用经验：按命中次数和有效伤害结算，单次匪巢建议 8-24 点

### 5.2 技能伤害与回复

- 外功技能基础口径：`技能倍率 x 外攻 + 固定值`
- 内功/暗器口径：`技能倍率 x 内攻 + 固定值`
- 回复技能口径：`基础回复 x (1 + 治疗效率) + 内攻/外攻系数`
- 护盾技能口径：`最大气血系数 + 攻击属性系数`

### 5.3 构筑推荐区间

| 场景 | 推荐战力 | 推荐耐久预算 | 推荐护命投入 |
| --- | --- | --- | --- |
| T1-T2 匪巢 | 220-360 | 低 | 无 |
| T3-T4 匪巢 | 380-620 | 中 | 1 个槽位可选 |
| T5 匪巢 | 650-900 | 中高 | 1-2 个槽位建议 |
| 深层密报/世界 Boss | 920+ | 高 | 2 个槽位建议 |
