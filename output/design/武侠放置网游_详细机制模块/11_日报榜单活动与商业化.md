# 11 日报榜单活动与商业化

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 用日报、榜单、活动和商业化边界，承接玩家的每日反馈、身份认同和长期运营节奏。
- 明确每周与每月活动的参与方式、奖励池、结算逻辑和公平边界。
- 确保商城只卖外观与便利，不破坏成长、经济和 PvP 公平性。

## 2. 解锁与入口

| 功能 | 解锁章节 | 入口 | 说明 |
| --- | --- | --- | --- |
| 江湖日报 | 第一章完成后 | 首页/日报页 | 每日首次登录生成 |
| 榜单 | 第五章 `黑风榜起` | 江湖页/榜单页 | 身份认同与荣誉展示 |
| 每周活动 | 第六章起 | 活动页/日报增刊 | 帮派和个人均可参与 |
| 每月世界事件 | 第八章起 | 活动页/地图页 | 与危险区和 Boss 联动 |
| 商城 | 第一章后 | 商城页 | 仅售外观与便利 |

## 3. 核心循环

`登录查看日报 -> 读取个人总结、帮派简报、子市场异动与热点路线 -> 进入当日目标 -> 参与榜单、活动和世界事件 -> 收取奖励 -> 用外观与便利变现，但不改变核心成长上限`

运营层的首发职责：

- 给玩家每天一个“今天干什么”的提示。
- 给不同玩法身份一个上榜和被看见的理由。
- 给项目稳定的收入线，但绝不卖直接数值。

### 4.6 奖励池口径

| 奖励档 | 内容 | 边界说明 |
| --- | --- | --- |
| 常规档 | 银两、声望、帮贡、基础材料 | 可稳定发放 |
| 进阶档 | 蓝紫装材、残页匣、功能外观 | 周活动主奖 |
| 稀有档 | 橙材、遗物部件、称号、增刊头条 | 仅世界事件和高排名 |

### 4.7 公平声明口径

- 不卖直接战斗力。
- 不卖经验翻倍、产出翻倍、免掉落、免损毁。
- 不卖额外离线时长。
- 可售便利必须明确“不提高核心产出上限”。
- 付费行具只扩功能槽，不扩战斗预算。

## 5. 公式与数值

### 5.1 日报生成

- 服务端每日 05:00 生成世界版素材
- 玩家当日首次登录时拼装个人版
- 重大事件触发“增刊”，同日可额外弹出 1 次

### 5.2 榜单结算建议

| 榜单 | 结算频率 |
| --- | --- |
| 武林榜 | 周结 |
| 富甲榜 | 周结 + 月度回顾 |
| 铸匠榜 | 周结 |
| 通缉榜 | 日更、周结 |
| 帮会战绩榜 | 周结 |

### 5.3 活动奖励建议区间

- 每周活动前 10%：进阶档 + 称号碎片
- 每周活动中位参与者：常规档 + 少量进阶档
- 每月事件首通：稀有档
- 每月事件参与保底：常规档
