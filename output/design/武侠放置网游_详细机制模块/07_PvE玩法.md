# 07 PvE玩法

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 形成首发主动内容和挂机内容的完整梯度，让玩家从匪巢簇一路打到密报链、组队副本和世界 Boss。
- 通过命名精英链、密报簇、公共痕迹和世界事件，让 PvE 同时服务成长、经济和 PvP 情报。
- 给首发 PvE 直接产出表、掉落池、节点簇规则、时限规则和前台展示文案。

## 2. 解锁与入口

| 功能 | 解锁章节 | 入口 | 说明 |
| --- | --- | --- | --- |
| T1-T2 匪巢簇 | 第一章/第三章 | 行动页/江湖页 | 新手挂机入口 |
| T3 命名精英链 | 第五章 `黑风榜起` | 匪巢结算页/地图 | 掉率开始明显提升 |
| 初级密报簇 | 第六章 `雾中来客` | 密报信笺页 | 1-3 人远征入口 |
| 3 人副本 | 第五章 `黑风榜起` | 江湖页 | 主流装备过渡 |
| 5 人副本 | 第八章 `外原立旗` | 江湖页 | 高端日常 |
| 世界 Boss | 第六章起轮换，第九章事件 Boss | 江湖页/日报增刊 | 稀有材料与话题中心 |

## 3. 核心循环

`刷匪巢簇 -> 获得银两、残骸、装备与密报 -> 触发命名精英链与公共痕迹 -> 决定继续清剿、出售密报或组队远征 -> 副本和 Boss 提供稳定或事件性高价值产出 -> 回流到制造、市场和 PvP 消耗`

PvE 的职责分层：

- 匪巢簇：日常挂机和惊喜感。
- 密报簇：小彩票与高价值机会。
- 常规副本：稳定供给和组队学习。
- 世界 Boss：公共事件和全服关注点。

## 5. 公式与数值

### 5.1 匪巢簇结算

`银两赏金 = 模板基础值 x 区域收益系数 x 清剿效率系数`

`残骸数量 = 模板残骸组 x (1 + 生活技能附加) x 随机波动`

`装备品质权重 = 模板权重 x 命名精英链加成 x 世界事件加成`

### 5.2 公共痕迹

- 每清完一次匪巢节点生成 1 点痕迹
- 同簇连续清剿最大叠至 5 点
- 每点持续 12 分钟
- 切换区域或停 15 分钟后开始自然衰减

### 5.3 密报过期处理

- 过期后自动转 `残页匣`
- 深层密报保留 1 份世界事件痕迹，但不再可进入
