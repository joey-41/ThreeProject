# 04 物品与资产

> 主索引：[武侠放置网游_详细游戏机制文档.md](../武侠放置网游_详细游戏机制文档.md)

## 1. 模块目标

- 建立统一物品框架，承接装备、材料、货物、消耗品、机关、密报和特殊资产。
- 通过耐久、绑定、分解、修理与容器规则，把“获得资产”和“失去资产”都解释清楚。
- 保证玩家能直观看懂背包、仓库、战利品栏、货仓四种容器的差异。

## 2. 解锁与入口

| 功能 | 解锁章节 | 入口 | 说明 |
| --- | --- | --- | --- |
| 背包 | 第一章 `临渚初入` | 仓库页 | 即时获得物品进入背包 |
| 仓库 | 第二章 `炉火青蒲` | 仓库页 | 制造、采集主容器 |
| 战利品栏 | 第三章 `夜渡青蒲` | 江湖/仓库页 | 匪巢和副本战利品先进入此处 |
| 货仓 | 第三章 `夜渡青蒲` | 行具页/运输页 | 与行具绑定 |
| 分解与修理 | 第二章 `炉火青蒲` | 仓库页 | 锻造和奇门设施共享 |
| 图鉴与制造者名片 | 第五章 `黑风榜起` | 物品详情 | 形成收藏和商誉 |

## 3. 核心循环

`获取物品 -> 进入对应容器 -> 装备/使用/交易/分解/损毁 -> 回流材料与银两 -> 支撑下一轮战斗、制造与运输`

资产系统的核心不是“给物品”，而是“让物品有去处”：

- 装备进入战斗与损毁循环。
- 材料进入制造与市场循环。
- 货物进入运输与劫镖循环。
- 消耗品进入挂机、远征和 PvP 风险缓冲循环。

| 容器 | 基础容量 | 进入物 | 溢出后果 |

## 5. 公式与数值

### 5.1 绑定规则

| 类型 | 规则 |
| --- | --- |
| 未绑定 | 可自由交易、上架、邮寄 |
| 装备后绑定 | 首次穿戴后绑定角色 |
| 订单绑定 | 仅对当前订单有效，不可转移 |
| 角色绑定 | 护命符、活动便利品等，获取即绑定 |

### 5.2 分解规则

`分解产出 = 基础残片 x 品质系数 x 完整度系数`

品质系数建议：

- 白 `1.0`
- 绿 `1.2`
- 蓝 `1.5`
- 紫 `1.8`
- 橙 `2.3`
- 遗物 `3.0`

### 5.3 修理规则

- 轻修：恢复 20% 耐久，花费为物品估值 3%
- 常修：恢复 50% 耐久，花费为物品估值 7%
- 精修：恢复 100% 耐久并去除轻微裂损，花费为物品估值 12%

### 5.4 死亡后的物品流向

- 货物和战利品优先掉落
- 装备进入损毁或耐久下降判定
- 绑定心法不掉落，但可能扣耐久或触发减损
- 密报不受保险保护，死亡时有概率直接失效
